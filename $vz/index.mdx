# `$vz`

A global object. It is recommended to access the API through this object.

| Properties            |
| --------------------- |
| [`api`](#api)         |
| [`manager`](#manager) |
| [`modules`](#modules) |

## Properties

### `.api`

The plugin API.

**Type: [`ApiManager`](./ApiManager)**

### `.manager`

An object that contains Vizality's managers.

**Type: `ManagerManager`**

### `.modules`

Miscellaneous tools to aid in plugin development.

**Type: `ModuleManager`**
